Cual es mejor HTML o HTML5

¿Qué es mejor: HTML o HTML5?

Característica HTML (Versiones antiguas) HTML5 (Versión actual)
Soporte Multimedia Requería plugins externos como Adobe Flash Player o Silverlight. Soporte nativo con etiquetas como <audio> y <video>.
Estructura Semántica Se abusaba de la etiqueta <div> para todo. Introduce etiquetas semánticas claras (<header>, <footer>, <article>, <nav>).
Almacenamiento Local Limitado al uso de Cookies (poca capacidad y menos seguro). Usa Web Storage (LocalStorage y SessionStorage), mucho más rápido y con mayor capacidad.
Gráficos y Animaciones Imposible de hacer de forma nativa sin herramientas externas. Permite gráficos vectoriales (SVG) y dibujo en mapa de bits con la etiqueta <canvas>.
Adaptabilidad (Mobile) No estaba pensado para teléfonos inteligentes. Diseñado desde cero pensando en la compatibilidad con dispositivos móviles.
